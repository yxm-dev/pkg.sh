[pkg.sh](https://github.com/yxm-dev/pkg.sh) is a simple package builder written in pure shell and aimed to
provide standardization to pure bash projects, with configurations prescribed in a `pkgfile`.

The packages created with [pkg.sh](https://github.com/yxm-dev/pkg.sh) comes with `install.sh` and
`unistall.sh` scripts that install/uninstall the needed dependencies (defined in the `pkgfile`). The uninstall
process fully remove the package and the installed dependencies.
